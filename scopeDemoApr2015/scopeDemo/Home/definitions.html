definitions

  • Every object in javascript is an associative array whose keys are string. The {} is an object literal.
  • Compilers cannot reduce string literals. It is better to define key as a string within single quote, if the intent is to access the value using the key string literal later.
  • scope : refers to variable visibility
  • In javascript, there are only two scopes global and function scope.
  • variables in global scope come into existence when it is declared and are deleted when you close the page
  • Usually the local variable in function are deleted when the function is completed.
  • However, using closures the environment that was present when the inner function was declared, is maintained by the javascript engine and can be accessed when the closure executes
  • In AngularJS $scope is the glue between controllers and views
  • Recommended practise: controllers write to $scope and views will read from $scope and display it
  • $scope objects have hierarchial structure and there is one and only one $rootscope for each ng-app element and it is root.
  • Use Batarang to see the $scope
  • Data Binding is an automatic way of updating the view whenever the model changes, as well as updating the model whenever the view changes. This is awesome because it eliminates DOM manipulation from the list of things you have to worry about.
  • Controllers are the behavior behind the DOM elements. AngularJS lets you express the behavior in a clean readable form without the usual boilerplate of updating the DOM, registering callbacks or watching model changes.
  • Services, factory and providers are singletons. They are all functions.
  • Use factory to create objects. The create methods in factory object usually accept parameters which are application specific. The factory create methods returns the object that was created.
  • Use service for common functionality that can be called from multiple places.
  • Providers can be injected where needed. The angularJS library will call the $get method. For example the $http object will get injected when the angularJS library calls the $get method in the httpProvider.
